HTML adalah bahasa markup yang digunakan untuk membuat struktur dan konten dari halaman web. Ini terdiri dari serangkaian tag yang menentukan bagaimana elemen dalam halaman web harus ditampilkan.

JavaScript adalah bahasa pemrograman yang sangat populer dan penting dalam pengembangan web modern. Bahasa pemrograman ini digunakan untuk membuat halaman web yang interaktif, responsif, dan dinamis

Java adalah salah satu bahasa pemrograman yang paling populer di dunia. Bahasa pemrograman ini pertama kali dikembangkan oleh James Gosling dan timnya pada tahun 1991 di Sun Microsystems. Saat ini, Java digunakan secara luas dalam pengembangan perangkat lunak, aplikasi web, permainan video, dan banyak lagi.
